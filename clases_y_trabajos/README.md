# Clases y Trabajos — Módulo 4

Esta carpeta contiene material docente preparado para las clases del módulo.

## Contenido

| Archivo | Descripción |
|---|---|
| `guion_clase.md` | Guion detallado para el docente: explica cada momento de la clase, los conceptos técnicos y los comandos a mostrar. |
| `contenido_ppt.md` | Contenido estructurado diapositiva por diapositiva, listo para trasladar a una presentación PowerPoint (`.pptx`). |
| `ejercicio_packet_tracer.md` | Actividad práctica guiada para que los estudiantes configuren IPs en Cisco Packet Tracer. |

## Tema de la clase

**Configuración de direcciones IP en Cisco Packet Tracer**

- Red de trabajo: `192.168.1.0/24`
- Rango reservado para routers: `192.168.1.1` – `192.168.1.9`
- Rango para computadores y dispositivos finales: `192.168.1.10` en adelante

Introducción a redes — estudiantes con conocimientos básicos de informática.
